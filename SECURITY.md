# Security And Privacy

ClaimContractBench is a static research-resource snapshot and standard-library
Python command suite. It is not a hosted service, but public release safety still
matters.

## Please Report

- accidental inclusion of confidential paper text or private review notes;
- raw third-party data that should not be redistributed;
- local absolute paths, credentials, private links, or institution-only files;
- a command that writes outside the requested output directory;
- a release manifest entry that is not public-safe.

## Please Avoid Public Details When Needed

If the issue includes sensitive material, do not paste it into a public issue.
Use GitHub private vulnerability reporting if it is available for the
repository, or open a minimal public issue saying that a private release-safety
report is needed.

## Project Boundary

This repository does not make legal, ethics, deployment, or acceptance
decisions. Security and privacy reports should focus on the repository release
surface, command behavior, and accidental disclosure risk.
